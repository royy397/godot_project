## Description of the project
A simple interactive piano game built in Godot 4. You can click on the keys to play musical notes from C4 to C5, with each key lighting up in a random colour when pressed. The background is a blue sky that sits behind the piano keys.
## Instructions for use
Click on any of the white piano keys to play a note. Each key plays a different musical note from C to C an octave higher. You can click multiple keys one after another to play a melody. There are no other controls needed, just click and play
## How it works:
The project is built in Godot 4. It creates 8 piano keys (C4 to C5) dynamically using a key scene. Each key has an AudioStreamPlayer node that loads a .wav sound file matching the note name. When a key is clicked, it plays the sound, changes to a random colour, and animates with a small scale tween. The background is a blue ColorRect that fills the screen.
